## 🙋‍♂️스터디 방식

- 매주 온/오프라인 면접 진행
- 주차별 진행 프로세스
    - 과목 시작 주차 : 각자 주제 발표
    - 이후 주차 : 각자 주제 발표 + 질문 리스트 내 질문 활용한 이전 주차 전체 페이지에 대한 면접
        - ~ 목요일 : 모든 주제에 대한 질문 4개씩(각자 각 주제에 대한 질문 최소 1개)을 함께 질문 페이지에 적기 ( 매주 4개 주제 최소 16개 이상 질문 )
- 각 과목 종료 시 과목별 종합 면접
- 그림 최대한 많이 활용
- 3:1 다대일 면접
